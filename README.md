# 学生管理系统 - C++ 控制台应用程序

## How to run

### 方法一：使用 Docker（推荐）

```bash
# 构建镜像
docker-compose build

# 交互式运行（控制台输入必需）
docker-compose run --rm cpp-student
```

### 方法二：本地编译（VS2022）

1. 打开 Visual Studio 2022
2. 创建一个新的"控制台应用"项目（C++）
3. 将以下文件添加到项目中：
   - `cpp-student/Student.h`
   - `cpp-student/Student.cpp`
   - `cpp-student/main.cpp`
4. 生成并运行（Ctrl+F5）

### 方法三：命令行编译

```bash
# 进入 cpp-student 目录
cd cpp-student

# 使用 g++ 编译
g++ -o student_system main.cpp Student.cpp -std=c++17

# 运行程序
./student_system
```

## 题目内容

实验三与四  类和对象 


一、目的要求 
1.	理解面向对象的程序设计的特点； 
2.	学会构造函数的使用和函数重载； 
3.	学会对象指针的使用、对象的动态建立和释放； 
4.	理解类的封装性和信息隐藏、析构函数的作用。 


二、实验内容： 
1.	定义学生类，包括学号、姓名、年龄、性别、多门课程的成绩，学生分为两类，课程门数不同，并将课程成绩全部初始化为0；假设有n个学生，由键盘输入学生信息； 
2.	定义类成员函数，完成修改每个学生的信息，计算每个学生的平均成绩，按课程统计每门课程的平均成绩、最高分和最低分；判断每个学生是否有不及格的课程； 
3.	同时输出所有学生信息； 
4.	通过输入学生学号或姓名查询某学生的信息。 

三、实验要求： 
1.	利用头文件和多个源文件分别实现类的声明、类成员函数的定义和主函数； 
2.	用函数的重载功能实现每个学生平均成绩的计算； 
3.	从键盘输入学生数，为对象动态分配内存空间，并保证对象使用完毕时释放内存空间； 
4.	定义构造函数完成类成员变量的初始化； 
5.	按照平均成绩对所有学生进行排名，并按照名次输出所有学生的信息（可只输出学生学号、姓名、性别和平均成绩。 
6.	利用析构函数输出"本程序由+自己的姓名+编写"字样，如"本程序由张三编写" 
7.	兼顾程序的逻辑性、人性化设计，并减少冗余代码，（比如输入时提示输入内容、输出格式一目了然等 
附加要求如下：1.数据格式要求：学号应为10位阿拉伯数字，姓名不超过四个汉字，性别只能位男、女，成绩须在0-100之间，若输入不合法，应有提示。2.设置单独的成绩修改函数，支持单独修改某个学生的某个门课成绩。3.实现按不同方式输出信息，按学号升序，按学号降序，按平均成绩升序，按平均成绩降序。 
要求按照初学者水平编写，变量和函数命名不要太复杂，因为使用vs2022，要求语法一定要符合vs2022要求，否则编译失败，包括字符串全部使用英文书写。要求尽量少的引用库，最好只有iostream和string，逻辑尽量简单，使用的函数尽量基础（如cin，cout）。代码中不能出现中文

## 项目描述

这是一个基于 C++ 控制台的学生管理系统，演示了面向对象编程的概念，包括类、构造函数、析构函数、函数重载、动态内存分配和指针等。

### 功能特性

1. **学生信息管理**
   - 使用动态内存分配添加多个学生
   - 两种学生类型：类型1（5门课程）和类型2（3门课程）
   - 存储学号、姓名、年龄、性别和课程成绩

2. **数据验证**
   - 学号：必须为10位数字
   - 姓名：最多8个字符
   - 性别：仅支持 M（男）或 F（女）
   - 成绩：必须在0-100之间

3. **查询功能**
   - 按学号查询
   - 按姓名查询

4. **修改功能**
   - 修改学生基本信息
   - 单独修改某门课程成绩

5. **统计功能**
   - 计算单个学生平均成绩
   - 按课程统计（平均分、最高分、最低分）
   - 识别有不及格课程的学生（成绩 < 60）

6. **排序和排名**
   - 按学号排序（升序/降序）
   - 按平均成绩排序（升序/降序）
   - 显示学生排名列表

### 技术亮点

- **面向对象设计**：封装的 Student 类，包含私有数据成员和公有方法
- **构造函数重载**：默认构造函数和参数化构造函数
- **函数重载**：`calcAverage()` 函数使用不同的参数列表
- **动态内存管理**：使用 `new` 和 `delete` 管理成绩数组和学生数组
- **拷贝构造函数和赋值运算符**：正确实现深拷贝
- **析构函数**：释放内存并打印作者信息
- **输入验证**：对所有用户输入进行全面验证
- **最小依赖**：仅使用 `<iostream>` 和 `<string>` 库
- **跨平台 Docker**：支持 AMD64 和 ARM64 架构
